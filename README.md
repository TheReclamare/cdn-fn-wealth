- **$43.7 billion** accumulated surplus
- **437,245** on-reserve population covered
- Per capita calculations
- Searchable, sortable table interface

## Features

- **Search** by band name
- **Sort** any column (click headers) ‚Äî ascending/descending
- **Columns:** Band Name, Year, Cash, Accumulated Surplus, Reserve Population, Cash/Capita, Surplus/Capita

## What is Accumulated Surplus?

Accumulated Surplus is essentially **Net Equity: Assets minus Liabilities**. It represents the band's total financial position built up over time.
